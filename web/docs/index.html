Purpur Downloads API

/v1 - Get all projects

/v1/{project} - Get project info

/v1/{project}/{version} - Get version info

/v1/{project}/{version}/{build} - Get build info

/v1/{project}/{version}/{build}/download - Download build

/v1/{project}/{version}/latest - Get latest build info

/v1/{project}/{version}/latest/download - Download latest build
